### What is Sigil?

Sigil is security infrastructure for AI agents operating on Solana DeFi. It gives asset owners enforceable controls over what their AI agents can do with their funds — spending caps, protocol allowlists, instruction-level constraints, and full audit trails.

**Agent autonomy. Human control.**

### How it works

```
Transaction = [ validate_and_authorize | DeFi instruction | finalize_session ]
```

Every agent transaction is sandwiched between authorization and finalization. All three succeed atomically or all revert. The agent can execute on any DeFi protocol — Jupiter, Flash Trade, Drift, Marginfi — while the vault owner's policy controls are enforced on-chain.
